# Smart Leads Dashboard 🚀

A simple Lead Management Dashboard built using:

- React
- TypeScript
- Express
- Node.js

This project is beginner friendly and made without MongoDB or complex MERN concepts.


# Features

✅ Add Leads  
✅ Delete Leads  
✅ Search Leads  
✅ Filter Leads by Status  
✅ Pagination  
✅ Export Leads to CSV  
✅ Dark Theme UI  
✅ Responsive Design  
✅ Built with TypeScript  


# Tech Stack

## Frontend
- React
- TypeScript
- Axios
- CSS

## Backend
- Node.js
- Express
- TypeScript
